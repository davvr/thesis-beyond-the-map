# beyond-the-map

An automated, reproducible pipeline that translates Spanish CIS national barometers into province-level seat projections under D'Hondt.

This is the codebase for my Master's thesis (TFM) in the MSc Computational Social Science at Universidad Carlos III de Madrid.
